# Darlean Proposal Calculator

Интерактивное коммерческое предложение для запуска Darlean на рынке США.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация

Push в ветку `main` запускает `.github/workflows/deploy-pages.yml` и
публикует production-сборку через GitHub Pages.
